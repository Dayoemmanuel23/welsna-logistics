# Welsna Logistics

A modern logistics and freight management platform built with React, Node.js, Express and MongoDB.

## Features

- Shipment Management
- Shipment Tracking
- Tracking History Timeline
- Online Quote Request
- Contact Form
- Email Notifications
- Admin Dashboard
- Ocean Freight
- Air Freight
- Inland Transportation
- Mobile Responsive
- SEO Optimized

---

## Tech Stack

Frontend

- React
- Vite
- Tailwind CSS
- Axios
- React Router

Backend

- Node.js
- Express
- MongoDB Atlas
- Nodemailer
- JWT Authentication

---

## Installation

### Frontend

```bash
npm install

npm run dev
```

Backend

```
cd backend

npm install

npm run dev
```

Environment Variables

Frontend

```
VITE_API_BASE_URL=https://your-render-api.onrender.com/api
```

Backend

```
PORT=5000

MONGODB_URI=

JWT_SECRET=

EMAIL_HOST=

EMAIL_PORT=

EMAIL_USER=

EMAIL_PASSWORD=

EMAIL_FROM=

FRONTEND_URL=
```
